# Chimera Linux WSL

Microsoft Store packaging for [Chimera Linux](https://chimera-linux.org/) on WSL2.

Chimera is an independent rolling-release distribution: musl libc, the
LLVM toolchain, dinit as init, apk-tools for package management, and a
FreeBSD-derived userland (chimerautils).

**Install:** search "Chimera Linux WSL" in the Microsoft Store, or
get the sibling [chimera-wsl2-installer](https://github.com/t3m3d/chimera-wsl2-installer)
PowerShell installer.
